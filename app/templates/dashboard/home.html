    Dashboard Overview

    The Financial Dashboard provides a comprehensive view of your financial status including:

    • Total balance across all accounts
    • Income and expense summaries
    • Budget tracking and alerts
    • Recent transaction history
    API Endpoints

    Main Dashboard Data:

    GET /financial/dashboard

    Financial Summary Only:

    GET /financial/dashboard/summary

    Account Summaries:

    GET /financial/dashboard/accounts

    Transaction Summary:

    GET /financial/dashboard/transactions

    Budget Status:

    GET /financial/dashboard/budgets
    Tips
    • Click on any account, transaction, or budget to view details
    • Use the refresh button to update data
    • Export functionality saves data in CSV format
    • Budget bars change color based on spending (green < 75%, yellow 75-90%, red > 90%)
